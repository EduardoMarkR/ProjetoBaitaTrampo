## 📌 Sobre o HirePulse

O **HirePulse** é uma plataforma de alocação de freelancers sob demanda, que automatiza a conexão entre empresas e trabalhadores via WhatsApp.

A solução permite distribuir vagas, coletar respostas e organizar contratações em tempo real, reduzindo esforço operacional e aumentando a velocidade de preenchimento das vagas.

---

## 🎯 Objetivo

Implementar um sistema automatizado para:

* Distribuição de vagas via WhatsApp
* Coleta automática de respostas dos candidatos
* Organização da base de freelancers
* Redução de processos manuais
